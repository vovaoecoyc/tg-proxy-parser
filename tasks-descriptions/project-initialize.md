## Цель
- Создать приложение для парсинга прокси серверов и добавления их в телеграм аккаунт

## Контекст
- [Telegram API](https://core.telegram.org/api)
- Список прокси брать из файла all_proxies.txt в [этом](https://github.com/SoliSpirit/mtproto) репозитории

## Задача
- Создать веб приложение которое:
   - получает список прокси из файла all_proxies.txt(при инициализации страницы)
   - получает список только новых прокси из последнего коммита, который изменил файл all_proxies.txt в репозитории [mtproto](https://github.com/SoliSpirit/mtproto). Критерием "нового прокси" является git diff , в котором берем только те строки которые были добавлены или изменены
   - проверяет статус прокси сервера(досутпен или нет) и показывает индикацию в ui
   - у каждого элемента списка есть кнопка обновления статуса сервера(проверка его доступности)
   - есть общая кнопка, которая инициирет проверку доступности всех серверов из списка

## Ограничения
- Приложение должно быть в виде веб страницы

## Критерии готовности
- Открывается веб интерфейс со списком прокси
- Отображаются статусы прокси из списка
- Работает обновление статусов для серверов отображающихся в списке
